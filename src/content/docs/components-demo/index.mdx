# Code Preview Tabs Component Demo

This page demonstrates the functionality of the `CodePreviewTabs` component.

Below is a preview of an HTML file located at `public/components-demo/sample.html`.

ignore content between `<!-- ignoreStart -->` and `<!-- ignoreEnd -->`,

## JS File Example (Preview Not Available)

This will demonstrate a JS file where only the code view should be available.
First, let's create a dummy JS file.

```js title="public/components-demo/sample.js"
// start
function greet(name) {
  // ignoreStart
  // This is an internal comment for the JS file.
  // ignoreEnd
  console.log(`Hello, ${name}!`);
  // // ignore render 
  // This line is for testing the parser, but JS files don't render visually.
  return `Hello, ${name}!`;
}

greet("World");
// end

// This is outside the scope.
```
*(The above JS code block is just for illustration; the component will fetch `public/components-demo/sample.js`)*
