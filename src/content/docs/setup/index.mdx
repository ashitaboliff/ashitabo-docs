さあ、いよいよ「あしたぼホームページ」の開発を始める準備をしましょう！
ここでは、あなたのパソコンで開発を進めるために必要な環境を整える方法を説明します。

開発環境の構築方法はいくつかありますが、主に以下の3つの方法を紹介します。
それぞれの特徴を読んで、自分に合った方法を選んでみてください。

## どの方法を選べばいいの？

- **プログラミングが初めての方、手軽に始めたい方**:
    - Windowsをお使いなら「Node.jsで環境構築 (Windows)」
    - Linuxをお使いなら「Node.jsで環境構築 (Linux / WSL)」
    - Dockerに興味がある、または将来的に他のプロジェクトでもDockerを使うかもしれない方は「Dockerで環境構築」も良い選択です。

- **既に他の開発でNode.jsを使っている方**:
    - お使いのOSに合わせてNode.jsの環境構築方法を選んでください。

- **PC環境を汚したくない、複数のプロジェクトで異なるバージョンのNode.jsを使いたい方**:
    - 「Dockerで環境構築」がおすすめです。

もし迷ったら、まずは「Node.jsで環境構築 (Windows)」または「Node.jsで環境構築 (Linux / WSL)」から試してみるのが良いでしょう。
途中で分からなくなったり、うまくいかなかったりした場合は、遠慮なく質問してくださいね。
