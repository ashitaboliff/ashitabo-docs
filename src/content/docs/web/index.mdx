## 学習の進め方

-   **プログラミングが初めての方**: まずは「[TypeScriptの基礎](/web/typescript)」から読み進めるのがおすすめです。プログラミングの基本的な考え方に触れることができます。
-   **JavaScriptの経験がある方**: 「[Reactの基礎](/web/react)」や「[Next.jsの基礎](/web/nextjs)」から始めても良いでしょう。
-   **デザインに興味がある方**: 「[Tailwind CSSの基礎](/web/tailwind)」は、見た目を整える技術なので、ここから入るのも面白いかもしれません。

各ページでは、できるだけ実際のコード例とそのプレビューを提示するように心がけています。
`src/components/CodePreviewTabs.astro` コンポーネントを使って、コードとその実行結果（または見た目のプレビュー）をタブで切り替えて確認できるようにする予定です。

{/*

もし分からないことがあれば、遠慮なく[用語集](/getstart/term)を参照したり、開発チームのメンバーに質問したりしてくださいね。
