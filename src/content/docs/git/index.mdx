## なぜGitとGitHubを使うの？

-   **変更履歴の管理**: いつ、誰が、どこを、どのように変更したかをすべて記録できます。
-   **過去のバージョンへの復元**: 間違った変更をしてしまっても、簡単に過去の状態に戻せます。
-   **並行作業の実現**: 「ブランチ」という機能を使うことで、複数の機能を同時に、安全に開発できます。
-   **チームでの共同作業**: GitHubを使うことで、複数人が同じプロジェクトのコードを共有し、効率的に共同作業を進められます。
-   **コードレビュー**: プルリクエストという機能を通じて、変更内容を他の人に見てもらい、フィードバックをもらうことができます。

さあ、GitとGitHubをマスターして、快適な開発ライフを送りましょう！
