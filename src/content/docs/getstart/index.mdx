## 開発のモチベーション
あしたぼホームページは、開発者のわたべが先輩たちの開発した既存のホームページ、部室予約システムをモダンな技術で再構築し、かつその技術を使って新たな機能を追加していくことを目的としています。
このプロジェクトを通じて、以下のことを学ぶことができます。
-   **Next.js** と **TypeScript**, **React** の使い方
-   **Figma** を使ったデザインの実装
-   **Tailwind CSS** を使ったスタイリング
-   **Git** を使ったチーム開発の方法
-   **Docker** を使った開発環境の構築と運用
-   **AIエージェント** を使った開発の効率化

以上の技術を学ぶことで、現代のWeb開発に必要なスキルを身につけることができますが、普通に僕もTypeScript経験1年(2025年5月現在)の若造なのでまだまだ知らないことがたくさんあります。
このプロジェクト、このドキュメントを通じて、もしくは超えて、皆さんがたくさんの知識をつけ、かつこのプロジェクトが受け継がれていくことを願っています。

また、好きなようにガクチカやポートフォリオに使ってください。嘘もOK！
